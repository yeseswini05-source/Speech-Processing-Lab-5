# Speech Processing Lab 5
Filter Bank Design (Triangular & Mel Filter Bank)

 Objective

* Construct triangular filters
* Design multiple overlapping filters (filter bank)
* Implement Mel filter bank
* Understand FFT bin mapping and Mel scale

 Files

* `SpeechProcessinglab5(1).ipynb` → Implementation
* `speechprocessinglab5report.pdf` → Detailed report

How to Run

 Install dependencies:

pip install numpy matplotlib

 Run:

jupyter notebook SpeechProcessinglab5(1).ipynb

 Outputs

* Single triangular filter
* Multiple triangular filters
* Mel filter bank visualization
* Mel points graph
* Filter parameters (a, b, c)

 Key Concepts

* Triangular filter (piecewise function)
* Mel scale (human auditory perception)
* FFT bin mapping
* Overlapping filter banks

 Observations

* Filters overlap to ensure smooth frequency coverage
* Mel scale is non-linear in Hz
* Filter spacing uniform in Mel scale but not in frequency
* FFT bins are essential for digital signal processing
